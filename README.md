# 🍔 Sistema de Gestão de Hamburgueria

Aplicação desenvolvida em **C#** que simula o funcionamento de uma hamburgueria, permitindo o **cadastro de produtos**, **criação de pedidos** e **cálculo automático do valor total**.

O projeto foi criado com o objetivo de praticar **lógica de programação** e **programação orientada a objetos**, utilizando conceitos fundamentais do desenvolvimento em C#.

## 🚀 Funcionalidades

- Cadastro de produtos do cardápio
- Criação e gerenciamento de pedidos
- Cálculo automático do valor total do pedido
- Manipulação de dados utilizando **List<T>**
- Organização da lógica utilizando **classes e métodos**

## 🛠 Tecnologias utilizadas

- C#
- .NET
- Programação Orientada a Objetos (POO)
- Estruturas de dados com `List<T>`

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para reforçar conceitos essenciais de desenvolvimento em C#, incluindo **estruturação de código, organização da lógica de negócio e manipulação de dados em memória**.
