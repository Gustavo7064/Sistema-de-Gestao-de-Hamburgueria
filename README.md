# 🍔 Sistema de Pedidos - Hamburgueria (C#)

Este projeto consiste em um **sistema simples de gerenciamento de pedidos para uma hamburgueria**, desenvolvido em **C#** com o objetivo de praticar conceitos fundamentais de **programação orientada a objetos (POO)** e estruturação de aplicações.

A aplicação simula o funcionamento básico de um sistema de pedidos, permitindo registrar clientes, adicionar itens ao pedido, organizar os produtos escolhidos e calcular automaticamente o valor total da compra.

## 🚀 Funcionalidades

- Criação e gerenciamento de pedidos
- Adição de itens (hambúrgueres, bebidas ou complementos)
- Organização dos produtos dentro do pedido
- Cálculo automático do valor total
- Estrutura de histórico de pedidos
- Manipulação de dados utilizando listas

## 🛠 Tecnologias utilizadas

- **C#**
- **.NET**
- Programação Orientada a Objetos (POO)
- Estruturas de dados com **List**
- Lógica de negócios aplicada ao fluxo de pedidos

## 📚 Conceitos aplicados no projeto

Durante o desenvolvimento foram aplicados diversos conceitos importantes para o desenvolvimento backend:

- SCRUM
- Criação de **classes e objetos**
- Utilização de **métodos para manipulação de dados**
- **Encapsulamento** e organização da lógica de negócio
- Uso de **listas para armazenamento de itens**
- Separação de responsabilidades dentro da aplicação

## 🎯 Objetivo do projeto

O principal objetivo deste projeto é reforçar conhecimentos em **C# e desenvolvimento backend**, simulando um sistema real de pedidos e aplicando boas práticas de organização de código e modelagem de dados.
